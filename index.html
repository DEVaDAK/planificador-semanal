Configurar Google Calendar

Necesitás un OAuth Client ID de tipo Web application. En Google Cloud agregá como Authorized JavaScript origin la URL exacta donde publiques esta app, por ejemplo https://TU-USUARIO.github.io o tu dominio propio.

  1. Creá o elegí un proyecto en Google Cloud.
  2. Habilitá Google Calendar API.
  3. Configurá la pantalla de consentimiento OAuth.
  4. Creá un cliente OAuth tipo Web application.
  5. Copiá el Client ID y pegalo acá.
